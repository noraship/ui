# Tokens

L'identité « **Hélios Prune** » vit entièrement dans des CSS variables préfixées `--nora-*`
(`src/styles/nora.css`). Les composants ne référencent jamais une valeur brute — toujours un token.
Le thème **sombre est le défaut** ; le clair adouci s'active via `data-theme="light"` sur `<html>`
(utilise le toggle « Thème » de la toolbar pour voir cette page dans les deux mondes).

> Pour changer d'accent plus tard : `node scripts/derive-accent.mjs "#xxxxxx"` imprime les
> valeurs AA des deux thèmes à reporter dans `nora.css`.
